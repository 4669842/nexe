# nexe v2

Create a single executable from your node.js application.

### Requirements

* Python 2 (for node.js)
* Node >= 4.0

### How To Use

nexe is invocaked 2 ways.

Via CLI

```
nexe [OPTIONS]

... todo ...
```

or Via API.

```js
let nexe = new Nexe({ /* options */ });
```
